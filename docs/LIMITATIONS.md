# Limitations

- Controlled contradictions are deliberately inserted and do not describe an
  actual procurement.
- The proprietary canonical engine is not included. The public repository runs
  from frozen, sanitized Evidence artifacts.
- Bidder eligibility cannot be determined without a bidder profile.
- Public price data is dated, reference-only, and not an exact document-item
  match in this package.
- Calculated reference cost excludes unresolved candidates unless explicitly
  stated otherwise.
- The demo supports one fixed evaluation package, not a general upload system.
- A live GPT-5.6 call requires the evaluator's own API key and can vary within
  the strict schema. Cached mode is the reproducible judging baseline.
- The output is not an actual bid, contract cost, expected profit, or final
  procurement decision.
